## Migration releases

### **Requisitos**
- NodeJS
- Secret
- Usuário
- Senha

### **Configuração**
- faça o clone do projeto
- entre na pasta
- rode: ``npm install``
- rode: cp .env-example .env
- abra o arquivo .env e passe os devidos valores
- por fim rode: npm run start

#### Se der tudo certo, vai exibir no terminal todo o log.
